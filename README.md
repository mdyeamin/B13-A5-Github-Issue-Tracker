Q1: What is the difference between var, let, and const?

  Ans: var: old, function-scoped, can re-declare and reassign.
  let: block-scoped, can reassign, cannot re-declare in same block.
  const: block-scoped, cannot reassign, cannot re-declare. 

Q2: What is the spread operator (...)?

  Ans: The spread operator ... expands or unpacks elements.
  It is mainly used to:
  copy arrays
  merge arrays
  copy objects
  pass multiple values into functions

Q3: What is the difference between map(), filter(), and forEach()?

  Ans: map() creates a new array by changing each item.
  filter() creates a new array with only the items that match a condition.
  forEach() just runs a function on each item and does not return a new array.

Q4: What is an arrow function?

  Ans: An arrow function is a shorter way to write a function in JavaScript. It uses => and is often cleaner than a regular function.
